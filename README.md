# 🏦 Banking Management System (Java + JDBC + H2)

A console-based Banking Management System built using **Core Java**, **JDBC**, and **H2 Database**.

This project demonstrates database connectivity, CRUD operations, and transaction management using Java.

---

## 🚀 Features

- ✅ User Registration
- ✅ User Login
- ✅ Deposit Money
- ✅ Withdraw Money
- ✅ Check Account Balance
- ✅ Transaction History
- ✅ Persistent Database (H2 File Mode)

---

## 🛠 Technologies Used

- Java (JDK 17)
- JDBC
- H2 Database
- Git & GitHub
