# Banking Management System

A console-based Banking System built using Java, JDBC and H2 Database.

## Features
- User Registration
- Login System
- Deposit Money
- Withdraw Money
- Check Balance
- Transaction History

## Technologies Used
- Java
- JDBC
- H2 Database
- Git & GitHub

## How to Run

Compile:
javac -cp ".;h2-2.4.240.jar" *.java

Run:
java -cp ".;h2-2.4.240.jar" BankingSystem
